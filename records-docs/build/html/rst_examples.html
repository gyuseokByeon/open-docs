2.4. 列表¶

2.4.1. 有序列表¶

1. 枚举的类型有:

    a. 阿拉伯数组: 1, 2, 3 ...
    b. 大小写字母: A-Z 或 a-z
    c. 大小写罗马字母: I, II, III, iV, ... 或 i, ii, iii, iv ...
2. 使用 **#** 自动生成枚举序号, 可以从任一数字开始

    4. xxx
    #. yyy
    #. zzz

  1. 枚举的类型有:

    1. 阿拉伯数组: 1, 2, 3 ...
    2. 大小写字母: A-Z 或 a-z
    3. 大小写罗马字母: I, II, III, iV, ... 或 i, ii, iii, iv ...

  2. 使用 # 自动生成枚举序号, 可以从任一数字开始

    1. xxx
    2. yyy
    3. zzz

  • xxx

2.5.3. 文档测试块 (Doctest Blocks)¶

文档测试块是交互式的Python会话，以 >>> 开始，一个空行结束。

>>> print('This is a doctest block.')
This is a doctest block.

ResNet-50 Triplet      **80.7** **67.9**     **84.3**   **85.0**

ResNet-50 OIM          72.5     58.2         77.5       79.2
